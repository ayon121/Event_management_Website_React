<h1>
    About This Project
</h1>
<p>
    This  a social event management website. I made this for any event management agency. There are many kinds of event. Only logged in user can see event details. There is also offers for VIP user. There is also a contact form. This is mainly frontend. There is no serverside. 
</p>

    Features
</h1>
<p>
    1. Responsive <br>
    2. Login with Email & Password <br>
    3. Social Login(Google) <br>
    4. User Friendly <br>
    5. Animation<br>
    6. Firebase
</p>
